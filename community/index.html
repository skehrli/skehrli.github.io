Join the Community!

<p> The System Crafters community meets in a few places online in case you want to engage with other people about your favorite tools and techniques! </p> <h2><a id="chat-with-us" class="anchor" href="#chat-with-us">¶</a>Chat With Us</h2><div class="outline-text-2" id="text-orgb101f28"> <p> If you&rsquo;d like to chat with the community, you can join us both on IRC and Matrix! </p> <ul class="org-ul"> <li><b>IRC</b>: Join <code>irc.libera.chat</code> and join the <code>#systemcrafters</code> room (<a href="https://web.libera.chat/?channel=#systemcrafters">web chat</a>)</li> <li><b>Matrix</b>: Join the <a href="https://matrix.to/#/%23systemcrafters-space:matrix.org">System Crafters Space</a> on matrix.org</li> </ul> <p> I recommend using the Matrix space with the official <a href="https://element.io/">Element</a> client if you prefer a Discord or Slack style experience with chat history, embedded links, etc. You can read more details about using the Libera.Chat IRC bridge to Matrix on <a href="https://libera.chat/guides/faq#can-i-connect-with-matrix">this page</a>. </p> <p> If you&rsquo;d prefer to chat from inside of Emacs, the <a href="https://github.com/alphapapa/ement.el">ement.el</a> Matrix client by <a href="https://github.com/alphapapa">Alphapapa</a> is an excellent choice! I will soon be producing a video for how to use it. </p> <p> If you&rsquo;d like a more classic experience, try the <a href="https://en.wikipedia.org/wiki/Internet_Relay_Chat">IRC</a> channel! If you&rsquo;ve never used IRC before, check out the following videos: </p> <ul class="org-ul"> <li><a href="https://systemcrafters.net/chatting-with-emacs/irc-basics-with-erc/">Chatting in IRC using Emacs and ERC</a></li> <li><a href="https://systemcrafters.net/live-streams/june-04-2021/">Improving the IRC experience in ERC</a></li> </ul> </div> <h2><a id="friday-streams" class="anchor" href="#friday-streams">¶</a>Friday Streams</h2><div class="outline-text-2" id="text-org406bc0f"> <p> Join us <b>every Friday</b> for our weekly show <b><a href="../live-streams">System Crafters Live!</a></b> The stream is broadcast on both <a href="https://youtube.com/@SystemCrafters">YouTube</a> and <a href="https://twitch.tv/SystemCrafters">Twitch</a>. </p> <p> In these streams we usually experiment with new packages, configurations, and customizations to understand the pros and cons of using them. Other times we have philosophical discussions about software, computing, and productivity. It&rsquo;s always a good time! </p> <p> <a href="https://time.is/compare/1800_in_Athens">Click here</a> to see the stream time in your local time zone! </p> </div> <h2><a id="community-projects" class="anchor" href="#community-projects">¶</a>Community Projects</h2><div class="outline-text-2" id="text-orgd2fc2bc"> <p> We also have a growing set of community projects to which you can join and contribute! If you want to learn more about GNU Emacs and GNU Guix, contributing to one of these projects is a great way to do that. </p> <blockquote> <p> Have an idea for a community project? Send me an e-mail: <code>david</code> at <code>systemcrafters.net</code> </p> </blockquote> </div> <h3><a id="crafted-emacs" class="anchor" href="#crafted-emacs">¶</a>Crafted Emacs</h3><div class="outline-text-3" id="text-org157f7b2"> <p> <a href="https://github.com/SystemCrafters/crafted-emacs/">Crafted Emacs</a> is sensible base Emacs configuration created with the following goals: </p> <ul class="org-ul"> <li>Helps you craft your own Emacs configuration faster by providing pre-configured modules for common tasks</li> <li>The code of the configuration is a learning resource to help you understand Emacs Lisp better</li> <li>Easy to remove from your configuration once you&rsquo;ve finished crafting your own!</li> </ul> <p> This project is maintained by <a href="https://github.com/jeffbowman">Jeff Bowman</a>. You can read <a href="https://jeffbowman.writeas.com/">Jeff&rsquo;s blog</a> for more information about the project! </p> </div> <h3><a id="dotcrafterel" class="anchor" href="#dotcrafterel">¶</a>dotcrafter.el</h3><div class="outline-text-3" id="text-orgc3f5665"> <p> <a href="https://github.com/daviwil/dotcrafter.el">dotcrafter.el</a> is a project I started while creating the <a href="https://systemcrafters.net/learning-emacs-lisp/">Learning Emacs Lisp</a> video series. It is meant to help you manage your dotfiles using an Emacs package, including the job of &ldquo;tangling&rdquo; your <a href="https://systemcrafters.net/emacs-from-scratch/configure-everything-with-org-babel/">literate Org Mode configuration</a> to the necessary output files. </p> <p> This project has been dormant for a while, I&rsquo;d be happy if someone wanted to contribute and move it forward! </p> </div> <h3><a id="guix-installer-and-live-cd-images" class="anchor" href="#guix-installer-and-live-cd-images">¶</a>Guix Installer and Live CD Images</h3><div class="outline-text-3" id="text-org893d454"> <p> I&rsquo;ve used GitHub Actions to automatically produce up-to-date installation images and Live CDs for GNU Guix using the full Linux kernel from the <a href="https://gitlab.com/nonguix/nonguix">Nonguix</a> channel: </p> <ul class="org-ul"> <li><a href="https://github.com/SystemCrafters/guix-installer">https://github.com/SystemCrafters/guix-installer</a></li> <li><a href="https://github.com/SystemCrafters/guix-live-image">https://github.com/SystemCrafters/guix-live-image</a></li> </ul> <p> If you have ideas for how to improve these, please feel free to send a PR! </p> </div>
